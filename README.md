# 现代汉语虚词练习平台

面向中文教育专业本科生的自适应虚词学习系统，涵盖介词、助词、连词、语气词四大虚词类别。

## 功能特点

### 自适应难度机制
- **初始难度**：基础题
- **升级条件**：连续答对 3 道基础题 → 切换为进阶题
- **降级条件**：答错 2 道进阶题 → 切回基础题
- 同难度下题目随机抽取，已答题目不重复

### 完整练习流程
1. **开始前**：显示平台说明、难度规则、答题须知
2. **练习中**：
   - 单题展示（含题干、选项/判断）
   - 实时反馈（对错 + 简要解析）
   - 进度条显示
   - 暂停/重置功能
3. **结束后**：自动生成精细化学习测评报告

### 精细化测评报告
- **基础数据**：总答题数、正确率、各难度正确率
- **类别分析**：介词、助词、连词、语气词错误率
- **薄弱点定位**：按具体考点精准定位问题
- **学习建议**：针对薄弱点给出改进建议

## 题库数据格式

题目数据存储在 `js/questions.js` 文件中，每道题目包含以下字段：

```javascript
{
  "id": "q001",              // 唯一标识符
  "type": "single",          // 题目类型：single(单选)/multiple(多选)/judgment(判断)
  "level": "basic",          // 难度级别：basic(基础)/advanced(进阶)
  "category": "particle",    // 虚词类别：preposition/particle/conjunction/modal
  "tag": "的地得误用",        // 具体考点标签
  "stem": "题干内容...",      // 题目内容
  "options": [               // 选项（判断题可省略）
    {"key": "A", "text": "选项A"},
    {"key": "B", "text": "选项B"}
  ],
  "answer": ["A"],           // 正确答案（数组格式）
  "analysis": "解析内容..."   // 答案解析
}
```

### 虚词类别映射

| 类别代码 | 中文名 | 常见考点 |
|---------|--------|---------|
| `preposition` | 介词 | 对/对于、把字句、被字句、在/于用法 |
| `particle` | 助词 | 的地得误用、了/过/着时态、所字结构 |
| `conjunction` | 连词 | 虽然但是/即使也、因为所以、不但而且 |
| `modal` | 语气词 | 吗/呢/吧疑问语气、啊/呀感叹语气 |

## 替换/扩展题库

1. 打开 `js/questions.js` 文件
2. 找到 `QUESTIONS` 数组
3. 按照上述数据格式添加或修改题目
4. 如有新的考点标签，需要同步更新 `TAG_DESCRIPTIONS` 和 `STUDY_SUGGESTIONS` 对象

## 技术栈

- **框架**：Vue 3 (CDN版本，无需构建)
- **样式**：原生 CSS + CSS Variables
- **存储**：localStorage（前端临时存储）

## 浏览器兼容性

支持所有现代浏览器：
- Chrome 60+
- Firefox 60+
- Safari 12+
- Edge 79+

## 注意事项

1. 答题数据仅保存在本地浏览器中，清除浏览器数据会导致记录丢失
2. 建议在练习结束后及时保存测评报告
3. 题库内容可根据教学需要自行扩展
